# 社区独居老人照护风险预警与服务调度系统

## 项目简介

本系统面向城市社区中的独居、高龄、空巢、慢病或行动不便老人，是一个围绕社区照护流程建设的管理信息系统原型。

## 服务角色
- 老人/家属
- 社区网格员
- 社区医生
- 物业维修人员
- 志愿者
- 社区管理者

## 如何运行

1. 直接双击 `index.html` 在浏览器中打开
2. 无需安装任何依赖，无需启动服务器
3. 所有数据存储在浏览器 localStorage 中

## 核心业务闭环

```
老人/家属提交报平安或求助
→ 系统自动进行智能辅助识别
→ 生成风险等级、事件类型和建议处置
→ 网格员人工复核
→ 系统生成服务工单
→ 推荐社区医生/物业/志愿者/网格员等责任人
→ 协同人员处理并反馈结果
→ 家属评价
→ 管理端查看统计看板和服务复盘数据
```

## 嵌入式 AI 功能说明

系统内置基于规则引擎的 AI 风险识别模块，实现：
- 文本分类（事件类型识别）
- 风险评分（低/中/高风险判定）
- 触发原因生成
- 派单推荐（责任人 + 处理时限）
- AI 置信度模拟
- 可解释规则说明（状态规则、设备规则、文本规则、分类规则）

**重要**：所有 AI 结果必须经过人工复核。详见 `AI_FUNCTION_DEMO.md`。

## 操作流程

1. 双击打开 `index.html`
2. 在系统工作台首页查看关键指标和待处理事项
3. 点击"老人/家属端"进入报平安/求助提交
4. 可点击示例按钮快速填充测试数据
5. 提交后系统自动进行 AI 风险识别
6. 进入"智能风险识别"查看 AI 分析结果
7. 进入"网格员复核"进行人工审核确认
8. 生成工单后进入"工单调度"确认派单
9. 进入"协同处理"接单并完成工单
10. 进入"管理驾驶舱"查看统计指标和复盘建议
